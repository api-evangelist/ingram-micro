# Ingram Micro

Ingram Micro Inc. is the world's largest technology distributor, providing a full spectrum of global technology and supply chain services to businesses worldwide. The company offers IT products, cloud solutions, and supply chain management services, connecting technology vendors with resellers and retailers globally.

## APIs

### Ingram Micro Reseller API

The Ingram Micro Reseller API provides REST API access to product catalog, pricing, discounts, stock levels, order management, and order status for IT resellers. It uses OAuth 2.0 authentication and provides a sandbox environment for testing.

- **Human URL:** https://developer.ingrammicro.com/reseller/api-documentation/United_States
- **Tags:** IT Distribution, Reseller, Catalog, Pricing, Orders
- **Properties:**
  - [Documentation](https://developer.ingrammicro.com/reseller/api-documentation/United_States)
  - [OpenAPI](openapi/ingram-micro-reseller-api-openapi.yml)

### Ingram Micro Vendor API

The Ingram Micro Vendor API provides REST API access for technology vendors to scale their business through Ingram Micro's distribution network, with access to inventory management, order processing, and supply chain integration.

- **Human URL:** https://developer.ingrammicro.com/vendor/api-documentation
- **Tags:** IT Distribution, Vendor, Supply Chain, Inventory
- **Properties:**
  - [Documentation](https://developer.ingrammicro.com/vendor/api-documentation)
  - [OpenAPI](openapi/ingram-micro-vendor-api-openapi.yml)
